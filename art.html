Основы программирования

HTML

HTML (HyperText Markup Language) – это язык разметки гипертекста, который используется для создания структуры веб-страниц. Он является основой большинства сайтов в интернете и предоставляет возможность структурирования контента через использование тегов. HTML позволяет создавать заголовки, абзацы, списки, таблицы, формы и другие элементы, а также вставлять мультимедийные объекты, такие как изображения и видео.

Основные компоненты HTML

Структура документа

Каждый HTML-документ имеет определенную структуру, которая включает обязательные элементы:

Навигация по курсу

Теги Атрибуты Структура документа Заголовки Параграфы Списки Изображения Ссылки Видео

<!DOCTYPE html>

<html lang="ru">

<head>

<meta charset="UTF-8">

<title>Название страницы</title>

</head>

<body>

Содержимое страницы

</body>

</html>

Скопировано!
  • <!DOCTYPE html> – объявляет версию HTML, используемую в документе.
  • <html> – корневой элемент всего документа.
  • <head> – содержит метаданные страницы, такие как заголовок, кодировка символов и информация для поисковых систем.
  • <title> – определяет название страницы, которое отображается в браузере.
  • <body> – содержит основное содержание страницы, видимое пользователям.

Теги

Тег – это основной строительный блок HTML-документа. Каждый тег состоит из имени, заключенного в угловые скобки < > . Например:

<p>Это параграф</p>

Скопировано!

В этом примере p – это имя тега, который обозначает параграф текста. Большинство тегов имеют открывающий (<p>) и закрывающий (</p>) варианты, между которыми размещается содержимое элемента.

Некоторые теги являются одиночными и не требуют закрывающего тега, например:

<br> <!-- Перенесение строки -->

Скопировано!

Атрибуты

Атрибуты предоставляют дополнительную информацию о теге. Они добавляются внутри открывающего тега после имени тега и состоят из пары «имя-значение». Значение атрибута всегда заключается в кавычки. Например:

<a href https://example.com>Ссылка на сайт</a>

Скопировано!

Здесь href – это атрибут, который указывает URL-адрес ссылки.

Примеры основных элементов

Заголовки

Заголовки создаются с помощью тегов от h1 до h6, где h1 – самый важный заголовок, а h6 – наименее значимый.

<h1>Главный заголовок</h1>

<h2>Подзаголовок</h2>

<h3>Ещё менее важный заголовок</h3>

Скопировано!

Параграфы

Параграфы создаются с использованием тега p .

<p>Это параграф</p>

Скопировано!

Списки

Списки могут быть нумерованными ( ol) и ненумерованными ( ul). Элементы списка создаются с помощью тега li.

<ul>

<li>Первый пункт</li>

<li>Второй пункт</li>

</ul>

<ol>

<li>Первый шаг</li>

<li>Второй шаг</li>

</ol>

Скопировано!

Изображения

Для вставки изображений используется тег img с атрибутом src, указывающим путь к изображению.

<img src=" image.jpg>" alt=" Описание изображения>

Скопировано!

Ссылки

Гиперссылки создаются с помощью тега a и атрибута href, который указывает адрес ссылки.

<a href=" https://example.com">Перейти на сайт</a>

Скопировано!
