JawaScript

JavaScript (часто сокращенно как JS) — это мультипарадигменный язык программирования, который широко используется для разработки веб-приложений, игр, десктопных приложений и многого другого. JavaScript является стандартизированным языком ECMAScript и используется в различных контекстах, включая клиентские веб-браузеры (например, Chrome, Firefox, Safari), серверные среды (Node.js), мобильные платформы (например, React Native) и даже встраиваемые системы (например, Tessel).

Основные понятия и синтаксис

  1. 1.Скриптовые файлы:
  • Файлы с расширением .js содержат JavaScript-код. Они могут быть подключены к HTML-странице с помощью элемента <script>.
  1. 2.Прототипное наследование:
  • JavaScript использует прототипное наследование, где каждый объект наследует свойства и методы от своего прототипа.
  1. 3.Объекты и функции:
  • В JavaScript все является объектами, включая функции. Функции могут использоваться как объекты и как первые классы.
  1. 4.Модульность:
  • Модули позволяют организовывать код в отдельные блоки, что улучшает структуру и читаемость.
  1. 5.АрROW-функции:
  • АрROW-функции упрощают создание анонимных функций и помогают улучшить читаемость кода.

Типы данных и операции

  1. 1.Primitive типы:
  • Типы данных, такие как boolean, number, string, null и undefined.
  1. 2.Object-types:
  • Объекты и массивы, которые могут содержать другие объекты или примитивные типы.
  1. 3.Операции:
  • Операции +, -, * и / для чисел и строк.
  • Условные (if), циклические (for, while) и логические (&&, ||) операции.

Объекты и прототипы

  1. 1.Объекты:
  • Используются для группировки данных и методов. Создаются с помощью литералов ( {} ) или вызова конструкторов.
  1. 2.Прототипы:
  • Каждый объект имеет свой прототип, который определяет его свойства и методы. Прототипы создаются с помощью оператора Object.prototype.
  1. 3.getters и setters:
  • Методы для управления доступом к свойствам объектов.

Функции и замыкания

  1. 1.Функции:
  • Создаются с помощью ключевого слова function или стрелки ( =>).
  1. 2.Замыкания:
  • Функции, которые захватывают контекст внешней области видимости.
