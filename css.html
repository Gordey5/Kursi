CSS

CSS (Cascading Style Sheets) – это язык стилей, который используется для описания внешнего вида веб-страниц, созданных с использованием HTML. CSS позволяет управлять такими аспектами, как цвета, шрифты, размеры элементов, их расположение на странице и многое другое. В этой теории мы рассмотрим основы работы с CSS, включая синтаксис, селекторы, каскадность и наследование.

Основы синтаксиса

Основной блок кода CSS состоит из правил, каждое из которых включает:

  1. Селектор – определяет элемент или группу элементов, к которым будут применяться стили.
  2. Блок объявлений – содержит одно или несколько свойств и значений, заключённых в фигурные скобки {} .

Пример простого правила:

p {

color: red;

}

В этом примере p является селектором, который выбирает все элементы <p> на странице, а color: red; – объявление, которое устанавливает красный цвет текста для этих элементов.

Селекторы

Селекторы позволяют выбрать конкретные элементы на странице для применения стилей. Основные типы селекторов включают:

  1. Теговые селекторы: выбирают элементы по имени тега. Например, h1, div , a .

h1 { font-size: 24px;}

  1. 2.Классовые селекторы:выбирают элементы по классу. Класс указывается через точку . . Например, .my-class.

<div class=" my-class">...< /div>

.my-class { background-color: yellow;}

  1. 3.Идентификаторные селекторы: выбирают элементы по уникальному идентификатору (id), указанному через решётку # . Например, #my-id.

<div class=" my-id">...< /div>

Скопировано!

.my-id { text-align: center;}

Скопировано!
  1. 4.Псевдоклассы и псевдоэлементы: позволяют выбирать элементы на основании состояния или позиции. Например, :hover , ::before , ::after.

a:hover { text-decoration: underline;}

p:first-line { font-weight: bold;}

Каскадность и приоритет

Каскадность означает, что при наличии нескольких правил, применимых к одному и тому же элементу, будет использоваться то правило, которое имеет больший приоритет. Приоритет определяется следующими факторами:

  1. Специфичность: чем больше специфичных условий у селектора, тем выше его приоритет. Идентификаторы имеют самый высокий приоритет, затем идут классы, псевдоклассы и атрибуты, после них – теговые селекторы.
  2. Порядок объявления: если два правила имеют одинаковую специфичность, применяется последнее объявленное правило.
  3. Важность: использование ключевого слова !important повышает приоритет свойства над всеми остальными правилами.

p {

color: blue !important;

}

Наследование

Некоторые свойства CSS могут наследоваться от родительских элементов до дочерних. Это позволяет экономить код и упрощает управление стилями. Примеры таких свойств включают font-family, color, line-height.

<body style="font-family: Arial, sans-serif;">

<div>

<p>Этот абзац унаследует шрифт Arial.</p>

</div>

</body>

Единицы измерения

В CSS используются разные единицы измерения для задания размеров, отступов и других параметров:

  1. Абсолютные единицы: пиксели (px), дюймы (in), сантиметры (cm) и другие.
  2. Относительные единицы: проценты (%), em (размер текущего шрифта), rem (размер корневого элемента).

Анимация появления модального окна

@keyframes-open {

from { top: -300px; opacity: 0;}

Цвета

Цвета в CSS могут задаваться несколькими способами:

  1. Названия цветов: black, white, red и др.
  2. Шестнадцатеричные коды: #000000, #FFFFFF, #FF0000.
  3. RGB/RGBA: rgb(255, 0, 0) или rgba(255, 0, 0, 0.5) (для прозрачности).
  4. HSL/HSLA: hsl(0, 100%, 50%) или hsla(0, 100%, 50%, 0.5).

Псевдо-классы и псевдо-элементы

Псевдо-классы и псевдо-элементы позволяют добавлять стили к элементам на основании их состояния или положения в документе:

  1. Псевдо-классы: :hover, :active, :focus, :visited, :nth-child(), :not() и др.
  2. Псевдо-элементы: ::before, ::after, ::first-letter, ::first-line и др.

Медиа-запросы

Медиа-запросы позволяют адаптировать внешний вид страницы под разные устройства и разрешения экрана. Они часто используются для создания отзывчивых дизайнов.

@media(max-width: 768px) {

body {

font-size 14px;

}

}
